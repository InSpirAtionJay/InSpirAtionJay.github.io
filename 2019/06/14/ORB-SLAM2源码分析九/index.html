2. Frame 类成员函数说明

2.1 构造函数

  • 默认构造函数，建立一个空的Frame

  • 复制构造函数，mLastFrame = Frame(mCurrentFrame)，完全复制当前帧的成员变量

  • RGB-D 构造函数

  • Stereo 构造函数
  • Monocular 构造函数，利用灰度图，时间戳，特征提取器，词典，相机内参，相机矫正参数，基线，深度截断值来构造一帧图像

2.2 获取特征点对应的格子坐标，并判断该特征点是否在原图像中

2.3 将矫正过的特征点分配到各个网格中，判断矫正过的特征点在哪个网格中，并将其加入到对应的表格vector中

2.4 特征提取函数ExtractORB，调用初始化的特征提取器进行特征提取，flag = 0単目提取，flag = 1，双目提取

2.5 设置相机姿态，随后会调用 UpdatePoseMatrices() 来改变mRcw,mRwc等变量的值

2.6 根据Tcw计算mRcw、mtcw和mRwc、mOw（Twc）

2.7 判断一个点是否在当前帧的视野内，将该地图点投影到图像中然后计算观测方向夹角，预测在当前帧的尺度

2.8 找到在以x,y为中心，边长为2r的方形内且在[minLevel, maxLevel]的特征点，先遍历该圆域区间所属的边长为2r的格子区间内的所有特征点，再判断该点的尺度信息是否满足要求，再判断该特征点离中心点的距离是否满足半径要求，返回该区间的所有的特征点

2.9 计算当前帧的词包词典

2.10 调用OpenCV的矫正函数矫正orb提取的特征点

2.11 计算图像边界
