track()线程主函数详解

此时track()函数已经完成了初始化，mstate = OK，mpInitializer已经初始化，track()函数进入else分支

1. 正常初始化 mState==OK

1.1 检查并更新上一帧被替换的MapPoints，Local Mapping线程可能会修改最后一帧图像的地图点

1.2 根据参考关键帧进行跟踪 TrackReferenceKeyFrame()

1.3 根据恒速模型和最后一帧图像进行跟踪 TrackWithMotionModel()，如果恒速模型不成功，再根据参考关键帧进行跟踪

2. 初始化完成后，特征点跟踪丢失，mState！=OK,此时进行重定位

3. 跟踪完当前帧后，需要进行更新信息

3.1 更新当前帧的参考关键帧为当前参考关键帧

3.2 跟踪局部地图 详细分解参照源码分析二

3.3 跟踪局部地图完成，mstate = OK

3.4 更新Motion模型

计算mVelocity，计算当前帧相对于最后一帧的相对位姿

3.5 清楚VO matches 清除UpdateLastFrame中为当前帧临时添加的MapPoints

3.6 从地图中清除地图点

3.7 检测是否需要插入关键帧

3.8 删除那些在bundle adjustment中检测为outlier的3D map点

3.9 判断跟踪状态，如果跟踪丢失，则reset

3.10 保存上一帧图像的数据，往下传递跟踪

3.11 记录位姿信息，用于轨迹复现
